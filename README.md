# NLU Coursework

Two projects for the Natural Language Understanding module.

## Assignment 01

Corpus statistics on the NLTK Gutenberg corpus (Zipf's Law, TTR, MATTR),
N-gram language models with Laplace smoothing (unigram, bigram, trigram)
including perplexity and ancestral sampling, and a Hidden Markov Model
part-of-speech tagger on the Brown corpus with a from-scratch Viterbi
decoder.

See `Assignment 01/` for code and report.

## Assignment 02

Sanskrit-to-English neural machine translation on a small parallel corpus
(10K train / 1K dev / 1K test), with a full fine-tune of the AI4Bharat
IndicTrans2 1.1B checkpoint as the winning configuration.

See `Assignment 02/README.md` for the winner metrics, layout, and setup.
